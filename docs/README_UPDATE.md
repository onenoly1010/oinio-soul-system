### üöÄ Getting Started with Pi Forge Integration

#### Prerequisites

- OINIO Soul System installed and running
- Python 3.7+ with pip
- Git for cloning the Pi Forge repository

#### Installation Steps

1. **Clone the Pi Forge Quantum Genesis repository:**
   ```bash
   git clone https://github.com/onenoly1010/pi-forge-quantum-genesis
   cd pi-forge-quantum-genesis
   ```

2. **Install Python dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure the integration path:**
   ```bash
   # Linux/macOS
   export PI_FORGE_PATH="/path/to/pi-forge-quantum-genesis"
   
   # Windows (PowerShell)
   $env:PI_FORGE_PATH = "C:\path\to\pi-forge-quantum-genesis"
   
   # Windows (Command Prompt)
   set PI_FORGE_PATH=C:\path\to\pi-forge-quantum-genesis
   ```

4. **Run OINIO with Quantum Enhancement:**
   ```bash
   ./oinio-system-linux
   ```

5. **Toggle Quantum Mode:**
   - In OINIO's soul menu, press **[Q]** to toggle quantum mode
   - When enabled, readings will include AI-enhanced predictions
   - Toggle back to **deterministic-only** mode anytime

### üõ†Ô∏è Troubleshooting

**Quantum mode not activating?**
- Verify `PI_FORGE_PATH` points to correct directory
- Check Python 3.7+ is installed: `python3 --version`
- Ensure `quantum_ai_enhancer.py` exists in Pi Forge directory
- Check OINIO console for Forge availability warnings

**Timeout errors?**
- Increase `QUANTUM_TIMEOUT_MS` (default: 3000ms)
- Verify Python dependencies are installed
- Check Pi Forge repository is complete (not partial clone)

**Want pure deterministic mode?**
- Simply don't install Pi Forge ‚Äî OINIO works standalone
- Or press **[Q]** in soul menu to toggle quantum mode off
- Set `ENABLE_QUANTUM=false` to disable integration
